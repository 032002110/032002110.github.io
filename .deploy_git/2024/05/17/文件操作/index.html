1. 文件管理基础

在iOS开发中，文件管理是一项常见的任务。Swift提供了多种方式来处理文件，包括创建、读取、写入和删除文件。本指南将介绍一些基本的文件操作方法。

1.1 文件路径

在iOS中，文件通常存储在沙盒目录中。常见的目录有：

  • Documents: 用户生成的文件可以存储在此目录中。
  • Library: 应用程序的配置文件和缓存文件可以存储在此目录中。
  • tmp: 临时文件可以存储在此目录中，系统可能会在应用退出后删除这些文件。
