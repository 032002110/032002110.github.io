然后又遇到了问题。。麻了最后兜兜转转还是先用csdn平台。csdn别倒闭了

1.5部署

采用一键部署的时候 我的setting pages选择的github actions 其中一些设置没调整好，最后使用了deploy from a branch 部署成功

上传本地源码用来多电脑协同博客

首先git init 本地文件
git remote add origin 这里origin是远程仓库名称 后面的是我们github的URL
git add. 添加文件到暂存区
git commit -m “sth” 提交暂存区的更改
git push -u origin HEAD:main
这里最开始push时候要求我 username 和password 我输了很多遍还是错误的，我没仔细看报错信息，还以为是我忘记了。其实报错信息是显示
这种验证方式从2021年后就淘汰了，由于我github URL是http形式的所以password提交应该是 token 。如果github URL是SSH就不要了
